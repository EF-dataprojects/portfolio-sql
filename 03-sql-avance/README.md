# Section 3 : SQL Avancé pour le Portfolio

Dans cette section, nous montrons des techniques avancées en SQL pour un portfolio :

- **Vues** pour visualiser les prêts des utilisateurs
- **Fonctions** et **procédures** pour automatiser les opérations
- **Triggers** pour loguer les actions
- Utilisation de **RANK**, **PERCENT_RANK** et calculs cumulés

Toutes les requêtes sont dans `queries.sql`.  
Les résultats simulés sont dans `results.md`.
